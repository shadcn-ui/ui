# 01. Primitives

Primitives are raw values with no product meaning. They are the vocabulary that semantic tokens reference.

## Colors

The initial paper source carries the minimum values needed to express the current v0 design target. Expand ramps here before adding new semantic tokens.

| Token | Value | Notes |
|---|---:|---|
| `primitives.colors.grey.0` | `#FFFFFF` | Elevated surface and contrast text. |
| `primitives.colors.grey.50` | `#F7F7F8` | Default app surface. |
| `primitives.colors.grey.100` | `#121215` | Highest-emphasis content. |
| `primitives.colors.grey.500` | `#494D5A` | Default content. |
| `primitives.colors.grey.700` | `#7E8395` | Muted content; not approved for normal text on white. |
| `primitives.colors.blue.600` | `#3237FF` | Primary interactive. |
| `primitives.colors.red.700` | `#F52512` | Alert feedback. |
| `primitives.colors.green.700` | `#149421` | Success feedback. |
| `primitives.colors.yellow.700` | `#A87900` | Warning feedback. |
| `primitives.colors.muted-blue.600` | `#707EB2` | Info feedback. |
| `primitives.colors.shift.400` | `#4B5B9F` | Shift accent. |
