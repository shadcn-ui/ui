# 04. Typography

Typography is modeled as role, size, and line-height. Paper source records the intended canonical values directly.

## Roles

| Token | Family | Weight | Tracking | Case |
|---|---|---|---:|---|
| `typography.roles.body` | Lead Sans Variable | Regular | 0 | original |
| `typography.roles.body-emphasis` | Lead Sans Variable | Medium | 0 | original |
| `typography.roles.title` | Lead Sans Variable | Regular | 0 | original |
| `typography.roles.title-emphasis` | Lead Sans Variable | DemiBold | 0 | original |
| `typography.roles.heading` | Lead Sans Variable | Regular | 0 | original |
| `typography.roles.display` | Lead Sans Variable | Regular | 0 | original |
| `typography.roles.caption` | Lead Sans Variable | DemiBold | 0 | original |
| `typography.roles.code` | Lead Sans Variable | Medium | 0 | uppercase |

## Sizes

| Token | Value | Notes |
|---|---:|---|
| `typography.sizes.code-xs` | 10 | `nonText`; badge/tag treatment only. |
| `typography.sizes.code-md` | 14 | Current code body size. |
| `typography.sizes.caption-sm` | 12 | Paper source starts at the text floor. |
| `typography.sizes.caption-md` | 12 | Default caption. |
| `typography.sizes.body-sm` | 14 | Data-dense body. |
| `typography.sizes.body-md` | 16 | Default body. |
| `typography.sizes.body-lg` | 18 | Large body. |
| `typography.sizes.title-sm` | 20 | Small title. |
| `typography.sizes.title-md` | 22 | Default title. |
| `typography.sizes.title-lg` | 25 | Large title. |
| `typography.sizes.heading-sm` | 28 | Small heading. |
