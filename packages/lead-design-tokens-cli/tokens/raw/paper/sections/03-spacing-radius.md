# 03. Spacing And Radius

Gap and padding are separate scales even when values match. Gap describes space between siblings; padding describes space inside a container.

## Gap

| Token | Value |
|---|---:|
| `spacing.gap.min` | 2 |
| `spacing.gap.2x-sm` | 4 |
| `spacing.gap.x-sm` | 8 |
| `spacing.gap.sm` | 12 |
| `spacing.gap.med` | 16 |
| `spacing.gap.lrg` | 20 |
| `spacing.gap.x-lrg` | 24 |
| `spacing.gap.2x-lrg` | 32 |
| `spacing.gap.max` | 48 |
