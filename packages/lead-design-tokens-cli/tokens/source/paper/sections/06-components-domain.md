# 06. Components And Domain

Component anatomy and domain tokens are intentionally deferred. The paper source keeps placeholders so the future namespaces have an obvious home without pretending the values are designed.

## Component Tokens

The current button collection is pre-anatomy. Full component tokens arrive later as:

`component.button.primary.{bg,fg,border}.{default,hover,active,disabled,focus}`
