shadcn/ui Button Component

Built with Leptos and Rust WASM

Basic Button

A simple button component rendered from Rust/WASM using Leptos framework.

Getting Started

1. Build the component

cd rust/components/button
wasm-pack build --target web --out-dir pkg

2. Serve this file

python3 -m http.server 8080
# Visit http://localhost:8080

3. Use in your HTML

<script type="module">
    import init, { mount_button } from './pkg/button.js';

    async function run() {
        await init();
        mount_button('#app', 'Click Me');
    }

    run();
</script>

Features

  • Built with Leptos reactive framework
  • Compiled to WebAssembly for performance
  • Styled with Tailwind CSS (matching shadcn/ui design)
  • Type-safe with Rust's type system
  • Small bundle size when compressed
  • Event handling support
