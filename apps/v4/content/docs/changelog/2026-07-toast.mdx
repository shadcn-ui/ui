A new [Toast](/docs/components/base/toast) component is now available for Base
UI projects. It supports actions, status types, promises, stacking, and swipe
dismissal.

<ComponentPreview styleName="base-nova" name="toast-demo" />

Add it to your project with the CLI:

```bash
npx shadcn@latest add toast
```
