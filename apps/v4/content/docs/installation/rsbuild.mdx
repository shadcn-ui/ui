## Use shadcn/create

<Steps>

### Build Your Preset

Open [shadcn/create](/create?template=rsbuild) and build your preset visually. Choose your style, colors, fonts, icons, and more.

<Button asChild size="sm">
  <Link
    href="/create?template=rsbuild"
    target="_blank"
    rel="noopener noreferrer"
    className="mt-6 no-underline!"
  >
    Open shadcn/create
  </Link>
</Button>

### Create Project

Click `Create Project`, choose your package manager, and copy the generated command.

The generated command will look similar to this:

```bash
npx shadcn@latest init --preset [CODE] --template rsbuild
```

### Add Components

Add the `Card` component to your project:

```bash
npx shadcn@latest add card
```

The command above will add the `Card` component to your project. You can then import it like this:

```tsx showLineNumbers title="src/App.tsx"
import {
  Card,
  CardContent,
  CardDescription,
  CardHeader,
  CardTitle,
} from "@/components/ui/card"

function App() {
  return (
    <Card className="max-w-sm">
      <CardHeader>
        <CardTitle>Project Overview</CardTitle>
        <CardDescription>
          Track progress and recent activity for your Rsbuild app.
        </CardDescription>
      </CardHeader>
      <CardContent>
        Your design system is ready. Start building your next component.
      </CardContent>
    </Card>
  )
}

export default App
```

</Steps>

## Use the CLI

<Steps>

### Create Project

Run the `init` command to scaffold a new Rsbuild project. Follow the prompts to configure your project.

```bash
npx shadcn@latest init -t rsbuild
```

### Add Components

Add the `Card` component to your project:

```bash
npx shadcn@latest add card
```

</Steps>

## Existing Project

<Steps>

### Create Project

If you need a new Rsbuild project, create one first with the React template. Otherwise, skip this step.

```bash
pnpm create rsbuild@latest
```

### Add Tailwind CSS

If your project already has Tailwind CSS configured, skip this step.

```bash
pnpm add -D tailwindcss @tailwindcss/postcss
```

Replace everything in `src/index.css` with the following:

```css title="src/index.css"
@import "tailwindcss";
```

### Configure PostCSS

Add a `postcss.config.mjs` file:

```js title="postcss.config.mjs"
export default {
  plugins: {
    "@tailwindcss/postcss": {},
  },
}
```

### Edit tsconfig.json file

If your project already has the `@/*` alias configured, skip this step.

```ts title="tsconfig.json" showLineNumbers
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["./src/*"]
    }
  }
}
```

### Update rsbuild.config.ts

Install `@types/node` and update `rsbuild.config.ts` so Rsbuild can resolve the `@` alias:

```bash
pnpm add -D @types/node
```

```typescript title="rsbuild.config.ts" showLineNumbers
import path from "path"
import { defineConfig } from "@rsbuild/core"
import { pluginReact } from "@rsbuild/plugin-react"

export default defineConfig({
  plugins: [pluginReact()],
  source: {
    alias: {
      "@": path.resolve(__dirname, "./src"),
    },
  },
})
```

### Run the CLI

Run the `shadcn` init command to set up shadcn/ui in your project:

```bash
npx shadcn@latest init
```

### Add Components

You can now start adding components to your project.

```bash
npx shadcn@latest add button
```

</Steps>
