```tsx showLineNumbers
  InputGroupText,
```

```tsx showLineNumbers
  <Input placeholder="Search..." />
  <InputGroupAddon>
    <SearchIcon />
  </InputGroupAddon>
</InputGroup>
```

## API Reference

### InputGroup

The main component that wraps inputs and addons.

| Prop        | Type     | Default |
| ----------- | -------- | ------- |
| `className` | `string` |         |

```tsx
<InputGroup>
  <Input />
  <InputGroupAddon />
</InputGroup>
```

### InputGroupAddon

Use the `InputGroupAddon` component to display icons, text, buttons, or other content alongside inputs.

<Callout icon={<IconInfoCircle />} title="Focus Navigation">
  For proper focus navigation, the `InputGroupAddon` component should be a
  placed after the input. Use the `align` prop to position the addon.
</Callout>

| Prop        | Type                                                             | Default          |
| ----------- | ---------------------------------------------------------------- | ---------------- |
| `align`     | `"inline-start" \| "inline-end" \| "block-start" \| "block-end"` | `"inline-start"` |
| `className` | `string`                                                         |                  |

```tsx
<InputGroupAddon>
  <SearchIcon />
</InputGroupAddon>
```

**For `<Input />
  `, use the `inline-start` or `inline-end` alignment. For `<Textarea />
  `, use the `block-start` or `block-end` alignment.**

### InputGroupButton

Use the `InputGroupButton` component to display buttons within input groups.

| Prop        | Type                                                                          | Default   |
| ----------- | ----------------------------------------------------------------------------- | --------- |
| `size`      | `"default" \| "icon" \| "sm" \| "icon-sm"`                                    | `"icon"`  |
| `variant`   | `"default" \| "destructive" \| "outline" \| "secondary" \| "ghost" \| "link"` | `"ghost"` |
| `className` | `string`                                                                      |           |

```tsx
<InputGroupButton>
  <CopyIcon />
</InputGroupButton>
```

### InputGroupText

Use the `InputGroupText` component to display text content within input groups.

| Prop        | Type     | Default |
| ----------- | -------- | ------- |
| `className` | `string` |         |

```tsx
<InputGroupText>(optional)</InputGroupText>
```
