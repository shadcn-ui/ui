## Usage

```tsx showLineNumbers
import {
  Timeline,
  TimelineConnector,
  TimelineContent,
  TimelineDescription,
  TimelineHeader,
  TimelineIcon,
  TimelineItem,
  TimelineTime,
  TimelineTitle,
} from "@/components/ui/timeline"
```

```tsx showLineNumbers
<Timeline>
  <TimelineItem>
    <TimelineConnector />
    <TimelineHeader>
      <TimelineIcon />
      <TimelineTitle>Event Title</TimelineTitle>
    </TimelineHeader>
    <TimelineContent>
      <TimelineDescription>Event description goes here.</TimelineDescription>
      <TimelineTime>Jan 15, 2024</TimelineTime>
    </TimelineContent>
  </TimelineItem>
</Timeline>
```

## API Reference

### Timeline

The root container for timeline items.

| Prop        | Type                               | Default      | Description                      |
| ----------- | ---------------------------------- | ------------ | -------------------------------- |
| orientation | `"vertical" \| "horizontal"`       | `"vertical"` | The orientation of the timeline. |
| position    | `"left" \| "right" \| "alternate"` | `"left"`     | Position of timeline items.      |

### TimelineItem

Individual timeline entry.

| Prop   | Type                                                  | Default     | Description                |
| ------ | ----------------------------------------------------- | ----------- | -------------------------- |
| status | `"default" \| "completed" \| "current" \| "upcoming"` | `"default"` | Visual status of the item. |

### TimelineIcon

The indicator dot/icon for each item.

| Prop    | Type                                                                  | Default     | Description                        |
| ------- | --------------------------------------------------------------------- | ----------- | ---------------------------------- |
| size    | `"sm" \| "default" \| "lg"`                                           | `"default"` | Size of the icon container.        |
| variant | `"default" \| "primary" \| "secondary" \| "destructive" \| "outline"` | `"default"` | Visual variant using theme colors. |

### TimelineConnector

The line connecting timeline items.

| Prop    | Type                                                  | Default     | Description                    |
| ------- | ----------------------------------------------------- | ----------- | ------------------------------ |
| variant | `"default" \| "dashed" \| "dotted"`                   | `"default"` | Line style.                    |
| status  | `"default" \| "completed" \| "current" \| "upcoming"` | `"default"` | Visual status affecting color. |

### TimelineTime

A semantic `<time>` element for timestamps.

| Prop     | Type     | Description                              |
| -------- | -------- | ---------------------------------------- |
| dateTime | `string` | Machine-readable date (ISO 8601 format). |
