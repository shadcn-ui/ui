## Usage

```tsx
import {
  Timeline,
  TimelineContent,
  TimelineDescription,
  TimelineItem,
  TimelineMarker,
  TimelineTime,
  TimelineTitle,
} from "@/components/ui/timeline"
```

```tsx
<Timeline>
  <TimelineItem>
    <TimelineMarker />
    <TimelineContent>
      <TimelineTitle>Event Title</TimelineTitle>
      <TimelineDescription>Event description goes here</TimelineDescription>
      <TimelineTime>2024-01-15 10:30 AM</TimelineTime>
    </TimelineContent>
  </TimelineItem>
</Timeline>
```

You can add custom icons to timeline markers using the `icon` prop.

  name="timeline-with-icons"

  description="Timeline with custom icons in markers"
  align="start"

```tsx
;<Timeline>
  <TimelineItem>

      variant="success"

```

```tsx

  <TimelineItem>
    <TimelineMarker />
    <TimelineContent>
```

Timeline with a centered vertical line and content alternating on both sides. Perfect for project roadmaps and process flows.

## Components

The Timeline component is composed of several sub-components:

- `Timeline` - The root container component
- `TimelineItem` - Individual timeline entry
- `TimelineMarker` - Visual indicator (dot or icon) with variant support
- `TimelineContent` - Content wrapper for each timeline item
- `TimelineSpacer` - Spacer element for center position layout
- `TimelineTitle` - Title/heading for the event
- `TimelineDescription` - Description text for the event
- `TimelineTime` - Time display element
