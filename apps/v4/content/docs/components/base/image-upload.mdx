## Examples

### Max files

Use `maxFiles` to limit the number of images. The drop zone is hidden once the limit is reached.

<ComponentPreview
  name="image-upload-max-files"
  previewClassName="p-6"
/>

### File type

Use `accept` to restrict which image types are allowed. Rejected files show an inline error.

<ComponentPreview
  name="image-upload-file-type"
  previewClassName="p-6"
/>

### Max size

Use `maxSize` (in bytes) to reject files that are too large.

<ComponentPreview
  name="image-upload-max-size"
  previewClassName="p-6"
/>

### Custom preview

Use `renderPreview` to replace the default thumbnail grid with your own layout. The render prop receives `files`, `previews`, `onRemove`, and `disabled`.
