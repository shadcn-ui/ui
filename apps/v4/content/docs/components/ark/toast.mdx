## About

The Toast component is built on top of the [Ark UI Toast](https://ark-ui.com/docs/components/toast) primitive.

## Usage

```tsx
import { toast } from "@/components/ui/sonner"
```

```tsx
toast.create({ title: "Event has been created." })
```
