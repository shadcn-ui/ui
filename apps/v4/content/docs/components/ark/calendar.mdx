## Usage

```tsx showLineNumbers
import { Calendar } from "@/components/ui/calendar"
```

```tsx showLineNumbers
const [date, setDate] = React.useState<Date | undefined>(new Date())

return (
  <Calendar
    mode="single"
    selected={date}
    onSelect={setDate}
    className="rounded-lg border"
  />
)
```

See the [Ark UI DatePicker](https://ark-ui.com/docs/components/date-picker) documentation for more information.

## About

The `Calendar` component is built on top of [Ark UI's DatePicker](https://ark-ui.com/docs/components/date-picker) primitive, which uses [`@internationalized/date`](https://react-spectrum.adobe.com/internationalized/date/) for date handling. The component bridges the `Date` API to Ark's `DateValue` internally.

## Date Picker

You can use the `<Calendar>` component to build a date picker. See the [Date Picker](/docs/components/ark/date-picker) page for more information.

## RTL

To enable RTL support in shadcn/ui, see the [RTL configuration guide](/docs/rtl).

<ComponentPreview
  styleName="ark-nova"
  name="calendar-rtl"
  direction="rtl"
  previewClassName="h-96"
/>

When using RTL, pass the `locale` prop as a string (e.g. `"ar-SA"`) and `dir="rtl"` to the Calendar component:

```tsx showLineNumbers
<Calendar
  mode="single"
  selected={date}
  onSelect={setDate}
  locale="ar-SA"
  dir="rtl"
/>
```
