## Usage

```tsx showLineNumbers
import { Calendar } from "@/components/ui/calendar"
import { CalendarDate, type DateValue } from "@internationalized/date"
import { type DatePickerValueChangeDetails } from "@ark-ui/react/date-picker"
```

```tsx showLineNumbers
const [value, setValue] = React.useState<DateValue[]>([
  new CalendarDate(2026, 3, 21),
])

return (
  <Calendar
    selectionMode="single"
    value={value}
    onValueChange={(details) => setValue(details.value)}
    className="rounded-lg border"
  />
)
```

See the [Ark UI DatePicker](https://ark-ui.com/docs/components/date-picker) documentation for more information.

## About

The `Calendar` component is built on top of [Ark UI's DatePicker](https://ark-ui.com/docs/components/date-picker) primitive with `inline` mode. It uses [`@internationalized/date`](https://react-spectrum.adobe.com/internationalized/date/) for date values — use `CalendarDate` for dates, `CalendarDateTime` for date+time, and `ZonedDateTime` for timezone-aware values.

## Date Picker

You can use the `<Calendar>` component to build a date picker. See the [Date Picker](/docs/components/ark/date-picker) page for more information.
