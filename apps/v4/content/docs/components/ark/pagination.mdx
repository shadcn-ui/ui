## Usage

```tsx showLineNumbers
import {
  Pagination,
  PaginationContext,
  PaginationEllipsis,
  PaginationItem,
  PaginationNext,
  PaginationPrevious,
} from "@/components/ui/pagination"
```

```tsx showLineNumbers
<Pagination count={100} pageSize={10} siblingCount={1}>
  <PaginationPrevious />
  <PaginationContext>
    {(api) =>
      api.pages.map((page, index) =>
        page.type === "page" ? (
          <PaginationItem key={index} {...page}>
            {page.value}
          </PaginationItem>
        ) : (
          <PaginationEllipsis key={index} index={index}>
            &#8230;
          </PaginationEllipsis>
        )
      )
    }
  </PaginationContext>
  <PaginationNext />
</Pagination>
```

## Changelog

### RTL Support

If you're upgrading from a previous version of the `Pagination` component, you'll need to apply the following updates to add the `text` prop:

<Steps>

<Step>Update `PaginationPrevious`.</Step>

```diff
  function PaginationPrevious({
    className,
+   text = "Previous",
    ...props
  })
```

<Step>Update `PaginationNext`.</Step>

```diff
  function PaginationNext({
    className,
+   text = "Next",
    ...props
  })
```

</Steps>
