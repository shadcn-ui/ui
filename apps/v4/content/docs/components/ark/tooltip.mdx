## Usage

```tsx showLineNumbers
import {
  Tooltip,
  TooltipContent,
  TooltipTrigger,
} from "@/components/ui/tooltip"
```

```tsx showLineNumbers
<Tooltip>
  <TooltipTrigger>Hover</TooltipTrigger>
  <TooltipContent>
    <p>Add to library</p>
  </TooltipContent>
</Tooltip>
```

Ark UI's Tooltip does not require a provider — it works out of the box.

## Examples

### Side

Use the `positioning` prop to change the placement of the tooltip.

```tsx showLineNumbers /positioning/
<Tooltip positioning={{ placement: "right" }}>
  <TooltipTrigger>Hover</TooltipTrigger>
  <TooltipContent>Right tooltip</TooltipContent>
</Tooltip>
```
