## Usage

The `Checkbox` component renders the control, indicator, and hidden input internally. Pass children to add a label.

```tsx
import { Checkbox } from "@/components/ui/checkbox"
```

```tsx
<Checkbox>Accept terms and conditions</Checkbox>
```

## Checked State

Use `defaultChecked` for uncontrolled checkboxes, or `checked` and `onCheckedChange` to control the state.

```tsx showLineNumbers
import * as React from "react"
import { Checkbox, type CheckboxCheckedChangeDetails } from "@/components/ui/checkbox"

export function Example() {
  const [checked, setChecked] = React.useState(false)

  return (
    <Checkbox
      checked={checked}
      onCheckedChange={(details: CheckboxCheckedChangeDetails) =>
        setChecked(!!details.checked)
      }
    >
      Accept terms
    </Checkbox>
  )
}
```
