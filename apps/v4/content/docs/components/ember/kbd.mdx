## API Reference

### Kbd

Use the `Kbd` component to display a keyboard key.

| Prop     | Type     | Default |
| -------- | -------- | ------- |
| `@class` | `string` | ``      |

```hbs showLineNumbers
<Kbd>Ctrl</Kbd>
```

### KbdGroup

Use the `KbdGroup` component to group `Kbd` components together.

| Prop     | Type     | Default |
| -------- | -------- | ------- |
| `@class` | `string` | ``      |

```hbs showLineNumbers
<KbdGroup>
  <Kbd>Ctrl</Kbd>
  <Kbd>B</Kbd>
</KbdGroup>
```
