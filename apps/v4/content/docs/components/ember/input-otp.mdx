## Usage

```gts showLineNumbers
import {
  InputOTP,
  InputOTPGroup,
  InputOTPSeparator,
  InputOTPSlot,
} from '@/ui/input-otp';
```

```hbs showLineNumbers
<InputOTP @maxLength={{6}}>
  <InputOTPGroup>
    <InputOTPSlot @index={{0}} />
    <InputOTPSlot @index={{1}} />
    <InputOTPSlot @index={{2}} />
  </InputOTPGroup>
  <InputOTPSeparator />
  <InputOTPGroup>
    <InputOTPSlot @index={{3}} />
    <InputOTPSlot @index={{4}} />
    <InputOTPSlot @index={{5}} />
  </InputOTPGroup>
</InputOTP>
```

## Examples

### Pattern

Use a custom pattern to validate the OTP input.

<ComponentPreview framework="ember" name="input-otp-pattern" />

The component uses a built-in pattern that only allows alphanumeric characters (letters and numbers).

### Separator

You can use the `<InputOTPSeparator />` component to add a separator between the input groups.

<ComponentPreview framework="ember" name="input-otp-separator" />

```gts showLineNumbers {4,15}
import {
  InputOTP,
  InputOTPGroup,
  InputOTPSeparator,
  InputOTPSlot,
} from '@/ui/input-otp';

<template>
  <InputOTP @maxLength={{4}}>
    <InputOTPGroup>
      <InputOTPSlot @index={{0}} />
      <InputOTPSlot @index={{1}} />
    </InputOTPGroup>
    <InputOTPSeparator />
    <InputOTPGroup>
      <InputOTPSlot @index={{2}} />
      <InputOTPSlot @index={{3}} />
    </InputOTPGroup>
  </InputOTP>
</template>
```
