## Usage

```gts showLineNumbers
import { Label } from '@/ui/label';
```

```hbs showLineNumbers
<Label @for="email">Your email address</Label>
```

## Label in Field

For form fields, use the [Field](/docs/components/ember/field) component which
includes built-in `FieldLabel`, `FieldDescription`, and `FieldError` components.

```gts showLineNumbers
import { Field, FieldLabel } from '@/ember-ui/field';
import { Input } from '@/ember-ui/input';
```

```hbs showLineNumbers
<Field>
  <FieldLabel @for="email">Your email address</FieldLabel>
  <Input id="email" />
</Field>
```
