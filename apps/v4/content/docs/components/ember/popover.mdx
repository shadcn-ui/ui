## Usage

```gts showLineNumbers
import {
  Popover,
  PopoverContent,
  PopoverTrigger,
} from '@/ui/popover';
```

```hbs showLineNumbers
<Popover>
  <PopoverTrigger>
    <Button @variant="outline">Open Popover</Button>
  </PopoverTrigger>
  <PopoverContent>
    Place content for the popover here.
  </PopoverContent>
</Popover>
```

## Composition

Use the following composition to build a `Popover`:

```text
Popover
├── PopoverTrigger
└── PopoverContent
```
