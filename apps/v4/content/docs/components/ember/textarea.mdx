```

```

## Examples

### Field

Use `Field`, `FieldLabel`, and `FieldDescription` to create a textarea with a label and description.

<ComponentPreview framework="ember" name="textarea-field" />

### Disabled

Use the `disabled` attribute to disable the textarea. To style the disabled state, add the `data-disabled` attribute to the `Field` component.

<ComponentPreview framework="ember" name="textarea-disabled" />

### Invalid

Use `aria-invalid="true"` to mark the textarea as invalid. To style the invalid state, add the `data-invalid` attribute to the `Field` component.

<ComponentPreview framework="ember" name="textarea-invalid" />

### Button

Pair with `Button` to create a textarea with a submit button.

<ComponentPreview framework="ember" name="textarea-button" />

### Variants

Use the `@variant` prop to change the visual style of the textarea.

<ComponentPreview framework="ember" name="textarea-variants" />

## API Reference

### Textarea

The `Textarea` component displays a multi-line text input field.

| Prop       | Type                                              | Default     |
| ---------- | ------------------------------------------------- | ----------- |
| `@variant` | `"outline" \| "filled" \| "underline" \| "ghost"` | `"outline"` |
| `@class`   | `string`                                          | -           |
