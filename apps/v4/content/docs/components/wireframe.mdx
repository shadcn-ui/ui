## Usage

```tsx showLineNumbers
import {
  Wireframe,
  WireframeNav,
  WireframeSidebar,
  WireframeStickyNav,
} from "@/components/ui/wireframe"
```

```tsx showLineNumbers
<Wireframe>
  <WireframeStickyNav>
    <div>My Sticky Navbar</div>
  </WireframeStickyNav>
  <WireframeNav position="top | bottom | responsive">
    <div>My Navbar</div>
  </WireframeNav>
  <WireframeSidebar collapsed="true | false" position="left | right">
    <div>My Sidebar</div>
  </WireframeSidebar>
  <main>Lorem Ipsum</main>
</Wireframe>
```

Note: If you have a responsive navbar, you must not have other navbars because responsive navbars ocuppy both top and bottom navbar spaces based on the screen width.

Sticky navbars can only be placed at the top so you can either have a sticky navbar or a fixed top navbar.
