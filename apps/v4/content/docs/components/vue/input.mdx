## API Reference

### Input

The `Input` component renders a styled HTML input element.

| Prop      | Type                                              | Default     |
| --------- | ------------------------------------------------- | ----------- |
| `variant` | `"outline" \| "filled" \| "underline" \| "ghost"` | `"outline"` |
| `type`    | `string`                                          | -           |
| `class`   | `string`                                          | -           |
