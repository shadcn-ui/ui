The `Item` component is a straightforward flex container that can house nearly any type of content. Use it to display a title, description, and actions. Group it with the `ItemGroup` component to create a list of items.

## Usage

```vue showLineNumbers
<script setup lang="ts">
import {
  Item,
  ItemActions,
  ItemContent,
  ItemDescription,
  ItemMedia,
  ItemTitle,
} from "@/components/ui/item"
</script>
```

```vue showLineNumbers
<template>
  <Item>
    <ItemMedia variant="icon">
      <Icon />
    </ItemMedia>
    <ItemContent>
      <ItemTitle>Title</ItemTitle>
      <ItemDescription>Description</ItemDescription>
    </ItemContent>
    <ItemActions>
      <Button>Action</Button>
    </ItemActions>
  </Item>
</template>
```

## Composition

Use the following composition to build an `Item`:

```text
ItemGroup
└── Item
    ├── ItemHeader
    ├── ItemMedia
    ├── ItemContent
    │   ├── ItemTitle
    │   └── ItemDescription
    ├── ItemActions
    └── ItemFooter
```

## Item vs Field

Use `Field` if you need to display a form input such as a checkbox, input, radio, or select.

If you only need to display content such as a title, description, and actions, use `Item`.

## API Reference

### Item

The main component for displaying content with media, title, description, and actions.

| Prop       | Type                                | Default     |
| ---------- | ----------------------------------- | ----------- |
| `variant`  | `"default" \| "outline" \| "muted"` | `"default"` |
| `size`     | `"default" \| "sm" \| "xs"`         | `"default"` |
| `as-child` | `boolean`                           | `false`     |

### ItemGroup

A container that groups related items together with consistent styling.

```vue
<template>
  <ItemGroup>
    <Item />
    <Item />
  </ItemGroup>
</template>
```

### ItemSeparator

A separator between items in a group.

```vue
<template>
  <ItemGroup>
    <Item />
    <ItemSeparator />
    <Item />
  </ItemGroup>
</template>
```

### ItemMedia

Use `ItemMedia` to display media content such as icons, images, or avatars.

| Prop      | Type                             | Default     |
| --------- | -------------------------------- | ----------- |
| `variant` | `"default" \| "icon" \| "image"` | `"default"` |

```vue
<template>
  <ItemMedia variant="icon">
    <Icon />
  </ItemMedia>
</template>
```

```vue
<template>
  <ItemMedia variant="image">
    <img src="..." alt="..." />
  </ItemMedia>
</template>
```

### ItemContent

Wraps the title and description of the item.

```vue
<template>
  <ItemContent>
    <ItemTitle>Title</ItemTitle>
    <ItemDescription>Description</ItemDescription>
  </ItemContent>
</template>
```

### ItemTitle

Displays the title of the item.

```vue
<template>
  <ItemTitle>Item Title</ItemTitle>
</template>
```

### ItemDescription

Displays the description of the item.

```vue
<template>
  <ItemDescription>Item description</ItemDescription>
</template>
```

### ItemActions

Container for action buttons or other interactive elements.

```vue
<template>
  <ItemActions>
    <Button>Action</Button>
  </ItemActions>
</template>
```

### ItemHeader

Displays a header above the item content.

```vue
<template>
  <Item>
    <ItemHeader>Header</ItemHeader>
    <ItemContent>...</ItemContent>
  </Item>
</template>
```

### ItemFooter

Displays a footer below the item content.

```vue
<template>
  <Item>
    <ItemContent>...</ItemContent>
    <ItemFooter>Footer</ItemFooter>
  </Item>
</template>
```
