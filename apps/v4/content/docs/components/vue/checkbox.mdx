## Usage

```vue showLineNumbers
<script setup lang="ts">
import { Checkbox } from "@/components/ui/checkbox"
</script>

<template>
  <Checkbox />
</template>
```

## Checked State

Use `:default-value` for uncontrolled checkboxes, or `:checked` and
`@checked-change` to control the state.

```vue showLineNumbers
<script setup lang="ts">
import { ref } from "vue"
import { Checkbox } from "@/components/ui/checkbox"

const checked = ref(false)
</script>

<template>
  <Checkbox :checked="checked" @checked-change="checked = $event" />
</template>
```

## Invalid State

Set `aria-invalid` on the checkbox and `data-invalid` on the field wrapper to
show the invalid styles.
