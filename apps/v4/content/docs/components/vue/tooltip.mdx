## Usage

```vue showLineNumbers
<script setup lang="ts">
import {
  Tooltip,
  TooltipContent,
  TooltipProvider,
  TooltipTrigger,
} from "@/components/ui/tooltip"
</script>

<template>
  <TooltipProvider>
    <Tooltip>
      <TooltipTrigger>Hover</TooltipTrigger>
      <TooltipContent>
        <p>Add to library</p>
      </TooltipContent>
    </Tooltip>
  </TooltipProvider>
</template>
```

## Composition

Use the following composition to build a `Tooltip`:

```text
TooltipProvider
└── Tooltip
    ├── TooltipTrigger
    └── TooltipContent
```
