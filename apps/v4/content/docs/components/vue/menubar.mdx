## Usage

```vue showLineNumbers
<script setup lang="ts">
import {
  Menubar,
  MenubarContent,
  MenubarGroup,
  MenubarItem,
  MenubarMenu,
  MenubarSeparator,
  MenubarShortcut,
  MenubarTrigger,
} from "@/components/ui/menubar"
</script>
```

```vue showLineNumbers
<template>
  <Menubar>
    <MenubarMenu>
      <MenubarTrigger>File</MenubarTrigger>
      <MenubarContent>
        <MenubarGroup>
          <MenubarItem>
            New Tab <MenubarShortcut>вҢҳT</MenubarShortcut>
          </MenubarItem>
          <MenubarItem>New Window</MenubarItem>
        </MenubarGroup>
        <MenubarSeparator />
        <MenubarGroup>
          <MenubarItem>Share</MenubarItem>
          <MenubarItem>Print</MenubarItem>
        </MenubarGroup>
      </MenubarContent>
    </MenubarMenu>
  </Menubar>
</template>
```

## Composition

Use the following composition to build a `Menubar`:

```text
Menubar
в”ңв”Җв”Җ MenubarMenu
в”Ӯ   в”ңв”Җв”Җ MenubarTrigger
в”Ӯ   в””в”Җв”Җ MenubarContent
в”Ӯ       в”ңв”Җв”Җ MenubarGroup
в”Ӯ       в”Ӯ   в”ңв”Җв”Җ MenubarLabel
в”Ӯ       в”Ӯ   в”ңв”Җв”Җ MenubarItem
в”Ӯ       в”Ӯ   в””в”Җв”Җ MenubarItem
в”Ӯ       в”ңв”Җв”Җ MenubarSeparator
в”Ӯ       в”ңв”Җв”Җ MenubarGroup
в”Ӯ       в”Ӯ   в”ңв”Җв”Җ MenubarLabel
в”Ӯ       в”Ӯ   в”ңв”Җв”Җ MenubarCheckboxItem
в”Ӯ       в”Ӯ   в””в”Җв”Җ MenubarCheckboxItem
в”Ӯ       в”ңв”Җв”Җ MenubarSeparator
в”Ӯ       в”ңв”Җв”Җ MenubarGroup
в”Ӯ       в”Ӯ   в”ңв”Җв”Җ MenubarLabel
в”Ӯ       в”Ӯ   в””в”Җв”Җ MenubarRadioGroup
в”Ӯ       в”Ӯ       в”ңв”Җв”Җ MenubarRadioItem
в”Ӯ       в”Ӯ       в””в”Җв”Җ MenubarRadioItem
в”Ӯ       в””в”Җв”Җ MenubarSub
в”Ӯ           в”ңв”Җв”Җ MenubarSubTrigger
в”Ӯ           в””в”Җв”Җ MenubarSubContent
в”Ӯ               в””в”Җв”Җ MenubarGroup
в”Ӯ                   в”ңв”Җв”Җ MenubarLabel
в”Ӯ                   в”ңв”Җв”Җ MenubarItem
в”Ӯ                   в””в”Җв”Җ MenubarItem
в””в”Җв”Җ MenubarMenu
    в”ңв”Җв”Җ MenubarTrigger
    в””в”Җв”Җ MenubarContent
        в””в”Җв”Җ MenubarGroup
            в”ңв”Җв”Җ MenubarLabel
            в”ңв”Җв”Җ MenubarItem
            в””в”Җв”Җ MenubarItem
```
