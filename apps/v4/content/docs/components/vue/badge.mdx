## Usage

```vue showLineNumbers
<script setup lang="ts">
import { Badge } from "@/components/ui/badge"
</script>

<template>
  <Badge variant="default | outline | secondary | destructive"> Badge </Badge>
</template>
```

## API Reference

### Badge

| Prop      | Type                                                                                                                                                                                     | Default     |
| --------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------- |
| `variant` | `"default" \| "secondary" \| "destructive" \| "outline" \| "ghost" \| "link" \| "warning" \| "success" \| "info" \| "success-solid" \| "warning-solid" \| "info-solid" \| "error-solid"` | `"default"` |

### Link

You can use the `as-child` prop to make another component look like a badge. Here's an example of a link that looks like a badge.

```vue showLineNumbers
<script setup lang="ts">
import { Badge } from "@/components/ui/badge"
</script>

<template>
  <Badge as-child>
    <a href="#">Badge</a>
  </Badge>
</template>
```
