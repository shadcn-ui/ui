## Customizing the icon

To change the icon size or spacing for a specific alert, simply pass a custom className to the icon or to the `AlertIcon` component:

```tsx
<Alert>
  <AlertIcon className="mr-4">
    <Terminal className="size-10 text-blue-600" />
  </AlertIcon>
  <AlertTitle>Custom icon size and spacing</AlertTitle>
  <AlertDescription>
    This alert uses a larger icon and extra spacing.
  </AlertDescription>
</Alert>
```

The `AlertIcon` component ensures that icon layout and spacing are easy to override per instance, without affecting other alerts or needing `!important`.
