## Usage

```tsx
import {
  ButtonGroup,
  ButtonGroupSeparator,
  ButtonGroupText,
} from "@/components/ui/button-group"
```

```tsx
<ButtonGroup>
  <Button>Button 1</Button>
  <Button>Button 2</Button>
</ButtonGroup>
```

## Accessibility

- The `ButtonGroup` component has the `role` attribute set to `group`.
- Use <Kbd>Tab</Kbd> to navigate between the buttons in the group.
- Use `aria-label` or `aria-labelledby` to label the button group.

```tsx showLineNumbers
<ButtonGroup aria-label="Button group">
  <Button>Button 1</Button>
  <Button>Button 2</Button>
</ButtonGroup>
```

## ButtonGroup vs ToggleGroup

- Use the `ButtonGroup` component when you want to group buttons that perform an action.
- Use the `ToggleGroup` component when you want to group buttons that toggle a state.

## Examples

### Orientation

The `ButtonGroup` component has an `orientation` prop that can be used to change the orientation of the button group.

<ComponentPreview name="button-group-orientation" />

```tsx showLineNumbers
import { ButtonGroup } from "@/components/ui/button-group"

export function ButtonGroupOrientation() {
  return (
    <ButtonGroup orientation="vertical">
      <Button>Button 1</Button>
      <Button>Button 2</Button>
    </ButtonGroup>
  )
}
```

### Size

The `ButtonGroup` component does not have a size prop, but you can control the size of the buttons within the group by using the `size` prop on the individual buttons.

<ComponentPreview name="button-group-size" />

```tsx showLineNumbers
import { IconPlus } from "@tabler/icons-react"

import { ButtonGroup } from "@/components/ui/button-group"

export function ButtonGroupSizes() {
  return (
    <ButtonGroup>
      <Button size="sm">Button 1</Button>
      <Button size="sm">Button 2</Button>
      <Button size="icon-sm">
        <IconPlus />
      </Button>
    </ButtonGroup>
  )
}
```

### Nested

Nest `<ButtonGroup>` components to create button groups with spacing.

<ComponentPreview name="button-group-nested" />

```tsx showLineNumbers
import { ButtonGroup } from "@/components/ui/button-group"

export function ButtonGroupNested() {
  return (
    <ButtonGroup>
      <ButtonGroup>
        <Button>Button 1</Button>
        <Button>Button 2</Button>
      </ButtonGroup>
      <ButtonGroup>
        <Button>Button 3</Button>
        <Button>Button 4</Button>
      </ButtonGroup>
    </ButtonGroup>
  )
}
```

### Separator

The `ButtonGroupSeparator` component is a separator that separates the buttons in the button group. Buttons with variant `outline` generally do not need a separator since they have a border. For other variants, a separator is recommended to improve the visual hierarchy.

<ComponentPreview name="button-group-separator" />

```tsx showLineNumbers
import { ButtonGroupSeparator } from "@/components/ui/button-group"

export function ButtonGroupSeparatorDemo() {
  return (
    <ButtonGroup>
      <Button>Button 1</Button>
      <ButtonGroupSeparator />
      <Button>Button 2</Button>
    </ButtonGroup>
  )
}
```

### Split

To create a split button group, add two buttons to the button group separated by a `ButtonGroupSeparator`.

<ComponentPreview name="button-group-split" />

```tsx showLineNumbers
import { IconPlus } from "@tabler/icons-react"

import { ButtonGroup, ButtonGroupSeparator } from "@/components/ui/button-group"

export function ButtonGroupSplitDemo() {
  return (
    <ButtonGroup>
      <Button>Button 1</Button>
      <ButtonGroupSeparator />
      <Button size="icon">
        <IconPlus />
      </Button>
    </ButtonGroup>
  )
}
```

## API Reference

### ButtonGroup

The `ButtonGroup` component is a container that groups related buttons together with consistent styling.

You can nest multiple button groups to create a more complex layout with spacing. See the [nested](#nested) example for more details.

| Prop          | Type                         | Default        |
| ------------- | ---------------------------- | -------------- |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` |

```tsx
<ButtonGroup>
  <Button>Button 1</Button>
  <Button>Button 2</Button>
</ButtonGroup>
```

### ButtonGroupSeparator

The `ButtonGroupSeparator` component is a separator that separates the buttons in the button group. Buttons with variant `outline` generally do not need a separator since they have a border. For other variants, a separator is recommended to improve the visual hierarchy.

| Prop          | Type                         | Default      |
| ------------- | ---------------------------- | ------------ |
| `orientation` | `"horizontal" \| "vertical"` | `"vertical"` |

```tsx
<ButtonGroup>
  <Button>Button 1</Button>
  <ButtonGroupSeparator />
  <Button>Button 2</Button>
</ButtonGroup>
```

### ButtonGroupText

Use this component to display text within a button group.

| Prop      | Type      | Default |
| --------- | --------- | ------- |
| `asChild` | `boolean` | `false` |

```tsx
<ButtonGroup>
  <ButtonGroupText>Text</ButtonGroupText>
  <Button>Button</Button>
</ButtonGroup>
```

You can use the `asChild` prop to render a custom component as the text, for example a label.

```tsx showLineNumbers title="Label in Button Group"
import { ButtonGroupText } from "@/components/ui/button-group"
import { Label } from "@/components/ui/label"

export function ButtonGroupTextDemo() {
  return (
    <ButtonGroup>
      <ButtonGroupText asChild>
        <Label htmlFor="name">Text</Label>
      </ButtonGroupText>
      <Input placeholder="Type something here..." id="name" />
    </ButtonGroup>
  )
}
```
