## About

`TileByWidget` (also exported as `DashboardTile`) renders pre-built demo widgets for dashboard layouts. It uses your existing `Card` and `Badge` components and [Recharts](https://recharts.org/) for charts.

## Usage

```tsx showLineNumbers
import { TileByWidget } from "@/components/ui/rgl-dashboard-tile"
```

```tsx showLineNumbers
<TileByWidget id="tile-kpi" type="kpi" />
```

Supported `type` values include `intro`, `kpi`, `chart`, `bar`, `table`, and `metric`.
