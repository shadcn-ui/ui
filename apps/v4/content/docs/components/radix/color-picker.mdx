## Usage

```tsx showLineNumbers

```

```tsx showLineNumbers
const [color, setColor] = React.useState("#ef4444")

return (
  <ColorPicker
    value={color}
    onChange={setColor}
    showShade
    className="w-64"
  />
)
```

## Props

| Prop               | Type                      | Default     | Description                               |
| ------------------ | ------------------------- | ----------- | ----------------------------------------- |
| `value`            | `string`                  | ŌĆö           | Controlled hex color value.               |
| `defaultValue`     | `string`                  | `"#ef4444"` | Initial color for uncontrolled usage.     |
| `onChange`         | `(value: string) => void` | ŌĆö           | Fires on every drag move.                 |
| `onChangeComplete` | `(value: string) => void` | ŌĆö           | Fires on pointer release.                 |
| `showShade`        | `boolean`                 | `false`     | Show the shade strip below the hue strip. |
| `disabled`         | `boolean`                 | `false`     | Disables all interaction.                 |
| `className`        | `string`                  | ŌĆö           | Additional class names on the wrapper.    |

## About

The `ColorPicker` component is built on top of [react-color-strip](https://www.npmjs.com/package/react-color-strip). It renders two strips ŌĆö a hue selector and an optional shade strip (enabled via `showShade`).

The component is **fully controlled or uncontrolled**. Pass `value` + `onChange` for controlled usage, or just `defaultValue` for uncontrolled.

## API Reference

See the [react-color-strip](https://github.com/aviralj02/react-color-strip#readme) documentation for the underlying strip primitives and pointer customization options.
