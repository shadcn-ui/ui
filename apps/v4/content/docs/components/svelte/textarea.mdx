## Examples

### Field

Use `Field`, `Field.Label`, and `Field.Description` to create a textarea with a label and description.

<ComponentPreview framework="svelte" name="textarea-field" />

### Disabled

Use the `disabled` prop to disable the textarea. To style the disabled state, add the `data-disabled` attribute to the `Field.Field` component.

<ComponentPreview framework="svelte" name="textarea-disabled" />

### Invalid

Use the `aria-invalid` prop to mark the textarea as invalid. To style the invalid state, add the `data-invalid` attribute to the `Field.Field` component.

<ComponentPreview framework="svelte" name="textarea-invalid" />

### Button

Pair with `Button` to create a textarea with a submit button.

<ComponentPreview framework="svelte" name="textarea-button" />

### Variants

Use the `variant` prop to change the visual style of the textarea.

<ComponentPreview framework="svelte" name="textarea-variants" />

## API Reference

### Textarea

The `Textarea` component displays a multi-line text input field.

| Prop      | Type                                              | Default     |
| --------- | ------------------------------------------------- | ----------- |
| `variant` | `"outline" \| "filled" \| "underline" \| "ghost"` | `"outline"` |
| `class`   | `string`                                          | -           |
