This directory contains the generated styles for the v4 registry. It is
autogenerated by the registry build process and gitignored — only this
README is tracked.

If the style directories are missing (e.g. on a fresh clone), generate them
with:

```bash
pnpm --filter=v4 registry:build --style all
```

This is the fast targeted build. A full `pnpm --filter=v4 registry:build`
also regenerates them, along with the installable registry JSON under
`public/r/styles`.
