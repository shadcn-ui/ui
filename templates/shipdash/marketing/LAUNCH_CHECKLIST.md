# Launch Checklist (Copy-Paste Ready)

## Pre-Launch (Day Before)

### Product Ready
- [ ] ZIP file tested and working (`./scripts/package.sh`)
- [ ] All pages load without errors
- [ ] Dark mode works
- [ ] Mobile responsive verified

### Gumroad Setup
- [ ] Product created
- [ ] ZIP uploaded
- [ ] Price set to $29
- [ ] Description copied from `GUMROAD-COPY.md`
- [ ] Screenshots uploaded (5-8 images)
- [ ] Product set to "Published"

### Marketing Ready
- [ ] Launch tweet drafted (copy from `TWITTER.md`)
- [ ] Reddit post drafted (copy from `REDDIT.md`)
- [ ] Indie Hackers post drafted (copy from `INDIE_HACKERS.md`)
- [ ] Screenshots saved in accessible folder

## Launch Day

### Hour 0: GO LIVE
- [ ] **Twitter:** Post launch tweet
- [ ] **Gumroad:** Verify product is live and purchasable

### Hour 1-2: Expand
- [ ] **Reddit r/SideProject:** Post (use copy from `REDDIT.md`)
- [ ] **Indie Hackers:** Post (use copy from `INDIE_HACKERS.md`)
- [ ] Reply to any comments on Twitter

### Hour 3-6: Engage
- [ ] Reply to EVERY comment and question

### End of Day
- [ ] **LinkedIn:** Post (use copy from `LINKEDIN.md`)
- [ ] Note down any FAQs for product page update
- [ ] Track sales count
